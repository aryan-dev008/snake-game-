## About

Classic Snake game built from scratch in Python using the Pygame library.
The snake eats food, grows longer, and dies if it hits a wall or itself.
Simple premise, addictive gameplay.

---

## Features

- 🐍 Smooth snake movement with gradient body coloring
- 🍎 Pulsing animated food
- 🧠 No-180° reverse turn (realistic control)
- 🏆 Persistent high score for the session
- 🎮 Keyboard support — Arrow keys + WASD
- 💀 Game over overlay with score display
- 🌑 Dark grid aesthetic

---

## Getting Started

**1. Clone the repo**
```bash
git clone https://github.com/aryan-dev008/snake-game-.git
cd snake-game-
```

**2. Install Pygame**
```bash
pip install pygame
```

**3. Run the game**
```bash
python snake_game.py
```

---

## Controls

| Key | Action |
|-----|--------|
| `↑` / `W` | Move Up |
| `↓` / `S` | Move Down |
| `←` / `A` | Move Left |
| `→` / `D` | Move Right |
| `Enter` / `Space` | Start / Restart |

---

## Scoring

- **+10 points** every time the snake eats food
- High score is tracked for the entire session
- Try to beat your own best!
